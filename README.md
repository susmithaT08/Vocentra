🎤 Vocentra – Where Communication Meets Voice

🌟 Overview

Vocentra is an intelligent communication enhancement platform designed to help users improve their speaking skills, confidence, and interaction abilities.
It combines voice-based analysis with AI-powered feedback to guide users toward better communication.

---

🚀 Features

- 🎙️ Voice-based communication practice
- 🧠 AI-powered feedback and evaluation
- 💬 Real-time conversation analysis
- 📊 Performance tracking and improvement insights
- 📚 Vocabulary and skill enhancement modules

---

🛠️ Tech Stack

Frontend:

- React / Next.js
- Tailwind CSS

Backend:

- Node.js
- Express.js

Other Tools & Services:

- Gemini API (AI processing)
- MongoDB / Database

---

⚙️ Installation & Setup

1️⃣ Clone the repository

git clone https://github.com/susmithaT08/Vocentra.git

2️⃣ Navigate to project

cd Vocentra

3️⃣ Install dependencies

npm install

4️⃣ Setup environment variables

Create a ".env" file and add:

PORT=5000
MONGO_URI=your_database_url
JWT_SECRET=your_secret_key

5️⃣ Run the project

npm start
